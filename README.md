# 🎓 Senior Connect - IIT Jodhpur

A premium directory platform designed to bridge the gap between juniors and seniors at IIT Jodhpur. Find mentors, explore tech stacks, and connect with experienced seniors seamlessly.

## 🚀 Live Demo
**[Visit Senior Connect Here](https://senior-connect-self.vercel.app/)**

## ✨ Features
* **Domain-Specific Mentorship:** Filter seniors by specific clubs or browse independent mentors based on their tech stack.
* **Verified Profiles:** Gamified "Blue Tick" verification system for seniors who complete their comprehensive profiles.
* **Automated Onboarding:** Integrated EmailJS invite system to directly onboard verified `@iitj.ac.in` users.
* **Secure Access:** Firebase Google OAuth strictly locked to official university email addresses.
* **Premium UI:** Custom glassmorphism, advanced Tailwind CSS animations, and seamless UX.

## 🛠️ Tech Stack
* **Frontend:** Next.js (React), Tailwind CSS
* **Backend as a Service:** Firebase (Firestore & Authentication)
* **Integrations:** EmailJS API
* **Deployment:** Vercel
